# Kader
A simple and responsive landing page for a kids' coding academy. Built with HTML and CSS, it includes a light/dark theme, project highlights, and a contact button. Also includes an Arabic version.
📚 Kader – Coding for Kids Landing Page
This is a responsive and colorful landing page for Kader, an educational platform focused on teaching programming and digital skills to children in a fun and engaging way.

🌟 Features
Clean and modern design using HTML and CSS

Light/Dark mode toggle

Sections for tools, student achievements, projects, and pricing plans

Arabic version available (arabic.html) for native users

Floating WhatsApp button for direct contact

Responsive layout for mobile and desktop

🚀 Technologies Used
HTML5

CSS3 (with variables and responsive grid)

Font Awesome for icons

Google Fonts (Cairo for Arabic readability)
